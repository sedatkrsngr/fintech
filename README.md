# Fintech Platform

Bu repository, ogrenme odakli ama production-grade hedeflenen bir fintech platformu icin monorepo temelidir.

## Calisma Sekli

- Projeyi adim adim kuruyoruz.
- Her adim once kisa teori, sonra uygulama, sonra ozet seklinde ilerleyecek.
- Sen `ogrendim` dediginde o adimi Git'e kaydedecegiz.
- Commit, senin onayin olmadan atilmayacak.

## Hedef

- Polyglot microservices architecture
- .NET 9 + Go + Next.js
- RabbitMQ + Kafka
- PostgreSQL + MongoDB + Redis + Elasticsearch
- YARP API Gateway
- Passwordless-ready auth
- Audit-first tasarim
- Dev / Test / Prod ortam ayrimi
- Production-grade CI/CD ve release-time migration

## Ilk Asama

Ilk asamada repo iskeleti, mimari kararlar ve temel proje standartlari kuruluyor.
